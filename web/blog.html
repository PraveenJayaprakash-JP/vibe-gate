Building Vibe Gate: A Quality Gate for AI-Generated Code

June 10, 2026

We built vibe-gate because we kept shipping broken auth flows, missing error handling, and untested edge cases in AI-generated web apps. The tool scans any deployed URL with Playwright, runs security and UX checks, and gives you a pass/fail report in under a minute.

The stack is Node.js + Playwright for the scanner, a lightweight Next.js dashboard for visualizing results, and a CLI that wraps it all into one command: npx vibe-gate https://your-app.com. The hardest part was making the checks deterministic across different frameworks — React, Vue, Svelte, or plain HTML all behave differently when it comes to routing, state hydration, and form submission.
