How it works

Three steps from URL to actionable report — no setup, no learning curve.

Run the command

npx vibe-gate https://your-app.com — that's it. Playwright launches, scans your app, collects data.

Get your grade

Color-coded A-F grade across 5 categories. Each check shows pass/warn/fail with specific findings.

Fix and ship

Plain-English recommendations tell you exactly what to fix. No jargon, no false positives.

What it checks

Every scan runs 15+ checks across 5 categories, giving you a complete picture of your app's health.

Page Health

Load success, HTTP status, render time, and title detection.

Console Errors

JavaScript errors, 404s, and API failures caught from the browser console.

Security Headers

CSP, HSTS, X-Frame-Options, and more — checks 6 critical security headers.

Secrets Detection

Scans for exposed API keys, tokens, and credentials in source code.

Auth Coverage

Detects unprotected routes, missing login pages, and auth gaps.

Local Code Scan

vibe-gate scan . checks your local project for issues before you deploy.

Everything you need to know about Vibe Gate.

Vibe Gate runs your URL in a headless Playwright browser and checks 5 categories: page health (does it load?), console errors (broken JS/API calls), security headers (CSP, HSTS, etc.), secrets exposure (leaked API keys), and auth coverage (unprotected routes). Each category gets a score, weighted by severity.
