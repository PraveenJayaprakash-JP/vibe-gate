How It Works

Three steps to quality

Run a scan, get your grade, fix issues before shipping.

1

Run the command

Type npx vibe-gate <url> in your terminal. No install, no config. Works on any URL or local file path.

2

Get your grade

Receive an A-F grade based on security, functionality, auth coverage, and visual consistency across your app.

3

Fix issues before shipping

Review the detailed report with pass/warn/fail breakdowns. Fix critical issues and re-scan until you hit grade A.

Features

Everything you need to ship with confidence

Comprehensive scanning powered by Playwright and static analysis.

Browser-based checks

Real Playwright browser automation tests your actual user flows, not just static HTML.

Security header scanning

Detects missing CSP, HSTS, X-Frame-Options, and other critical security headers.

Secret detection

Scans for exposed API keys, tokens, and credentials in your frontend code and network requests.

Auth coverage analysis

Identifies unprotected routes, missing auth redirects, and session handling gaps.

Mobile viewport testing

Validates responsive layouts, touch targets, and mobile-specific issues at multiple breakpoints.

Visual regression detection

Captures screenshots and detects layout shifts, broken images, and visual inconsistencies.

Local codebase scan

Point Vibe Gate at your local project folder to scan source files without deploying.

CI/CD integration

GitHub Action ready. Fail builds on grade D or below. Integrate with any CI pipeline in minutes.
