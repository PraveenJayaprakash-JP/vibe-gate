How it works

Three steps from zero to a clean bill of health.

1

Run

Point the CLI at any live URL. No config, no setup. It crawls every route automatically.

2

Grade

Get an instant letter grade across six categories, from auth to accessibility.

3

Fix

Follow the actionable report. Some fixes are one-liners; others get detailed guidance.

Everything you need to ship clean

Deep scans, clear grades, and actionable fixes — all from your terminal.

Security Scan

Detects missing headers, exposed secrets, insecure forms, and auth bypass paths before they hit production.

Route Audit

Crawls every page and checks for broken links, missing metadata, duplicate titles, and orphaned routes.

Auth Checks

Validates login flows, session handling, password rules, and protected-route enforcement end to end.

Performance

Measures load times, asset sizes, render-blocking resources, and Core Web Vitals at the route level.

Accessibility

Flags missing labels, low contrast, keyboard traps, and ARIA issues with WCAG-aligned severity.

CI/CD Ready

Exit codes, JSON output, and GitHub Actions support let you block merges on failing grades.
